# Yum-Chocolates-Sales-Analysis-Project-
This repository contains a Chocolate Sales Analysis Portfolio Project designed to demonstrate real-world data analytics skills including data cleaning, modeling, metric calculations, and building an interactive Power BI dashboard.
Project Overview

In this project, we analyze sales data from a fictional chocolate company to uncover trends, performance metrics, and business insights. The focus is on building a clean data model and creating a Power BI dashboard that helps stakeholders track sales performance, growth trends, and product success.

---

Objective

The goal of this analysis is to understand and visualize:
- Total chocolate sales by product and region
- Monthly performance trends
- Salesperson performance
- Key business metrics such as profit, cost, and shipments

---

Tools & Technologies Used

- **Power BI Desktop** – Dashboard creation & data visualization
- **Power Query** – Data transformation & modeling
- **DAX (Data Analysis Expressions)** – Metrics & KPI calculations
- **Excel / CSV** – Source data
- **GitHub** – Version control and portfolio hosting
---
   Workflow Summary

Data Import & Transformation
- Imported raw data into Power BI using Power Query
- Cleaned data fields including dates and costs
- Handled missing values and standardized categories
- Created relationships for data modeling

---

Data Modeling
- Built a star schema model
- Linked facts (sales) with dimensions (products, salesperson, regions)
- Ensured data consistency for accurate reporting

---

DAX Metrics & Calculations
Calculated key metrics using DAX, including:

dax: Total Sales = SUM('Sales'[SalesAmount])
Other metrics:
Profit
Total Boxes Shipped
Low Box Shipment %
Monthly trends

---
Dashboard Design & Visuals
Created interactive dashboards with:
Bar Charts – Show total shipment by product
Line Graphs – Monthly sales and trend analysis
KPI Cards – Total sales, profit, cost, shipments
Donut Charts – Regional sales distribution

---
Key Insights
✔ Top-selling chocolate products contribute heavily to revenue
✔ Monthly sales trends show seasonal demand peaks
✔ Certain regions outperform others in total shipment
✔ Top-performing salespersons drive revenue growth
